*dec.py*
```python
import gmpy2
p=473398607161
q=4511491
e=17
t=(p-1)*(q-1)
d=gmpy2.invert(e,t)
print(d)
```

> gmpy2.invert()是gmpy2库中的一个函数，用于求一个整数在模意义下的乘法逆元。具体来说，如果a和m是两个整数，且gcd(a, m) = 1，则gmpy2.invert(a, m)返回一个整数b，满足：
(a * b) % m = 1

私钥是公钥在模欧拉函数的乘法逆元
